# **App Name**: Sonho de Flor

## Core Features:

- Product Display: Showcase pajamas with images, descriptions, and prices.
- Banner Promotions: Display rotating promotional banners.
- Category Buttons: Include three category buttons to navigate the store. This includes future configuration options.
- Contact Information: Display the store name, contact email, WhatsApp number, phone number, Facebook link, and Instagram link.
- Navigation Bar: Provide a navigation bar with the store name, category buttons, and contact links.

## Style Guidelines:

- Primary color: Medium lavender (#9370DB) to evoke the lavender theme.
- Background color: Light lavender (#E6E6FA) to complement the primary color.
- Accent color: Darker lavender (#7B68EE) to add subtle contrast and highlight elements.
- Body and headline font: 'Alegreya', a serif font that gives an elegant, intellectual feel.
- Use sleep and lavender themed icons for visual appeal.
- Simple, grid-based layout to showcase the pajamas, with a rotating banner for new promotions at the top of the page.
